# Security policy

AstroProject Forge processes local astronomical files and can create large exported projects. Security and data-loss reports should not be disclosed in a public issue before a fix is available.

For now, contact the repository owner privately through the GitHub profile associated with this repository. Include the affected version, operating system, impact and minimal reproduction. Do not attach real acquisition data, personal paths, credentials or unredacted logs.

The public beta receives security fixes on the latest beta release only.
