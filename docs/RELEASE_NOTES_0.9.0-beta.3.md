# AstroProject Forge 0.9.0-beta.3

## English

### Interface language

- Italian and English are available on Windows, Linux and macOS.
- Change the interface at any time from **Settings → Language / Lingua**.
- The change is immediate and persists across launches.
- Navigation, workspaces, actions, status counters and tooltips follow the selected language.

### Downloads

The release page contains only installers and portable packages. GitHub supplies the source-code ZIP and TAR.GZ automatically. QA reports, manifests, checksums and SBOM files remain internal workflow artifacts.

### Signing status

Windows packages are not Authenticode-signed and macOS packages are not Apple-notarized yet. Trusted signing requires the project owner's private certificates and platform credentials.

## Italiano

### Lingua dell’interfaccia

- Italiano e inglese sono disponibili su Windows, Linux e macOS.
- La lingua si cambia in qualsiasi momento da **Impostazioni → Lingua / Language**.
- La modifica è immediata e resta salvata ai riavvii successivi.
- Navigazione, workspace, azioni, contatori di stato e tooltip seguono la lingua selezionata.

### Download

La pagina della release contiene soltanto installer e pacchetti portable. GitHub fornisce automaticamente gli archivi ZIP e TAR.GZ del sorgente. Report QA, manifest, checksum e SBOM restano artefatti interni della pipeline.

### Stato firma

I pacchetti Windows non sono ancora firmati Authenticode e quelli macOS non sono ancora notarizzati da Apple. La firma attendibile richiede certificati privati e credenziali di piattaforma del titolare.
