# AstroProject Forge 0.9.0-beta.1

## English

This is the first public beta of AstroProject Forge: a cross-platform calibration workspace for turning multi-night FITS/XISF acquisitions into auditable PixInsight WBPP projects.

### Included

- generic FITS/XISF import and header-first metadata analysis;
- astronomical-night reconstruction across midnight;
- filter → configuration-session → night project hierarchy;
- automatic Flat Epochs plus manual links across files, nights and sessions;
- explainable Flat/Dark/Bias matching with prioritized Master Libraries;
- acquisition statistics and CSV/JSON export;
- adaptive WBPP Grouping Keywords and verified project export;
- optional Quality Lab and independent Master Library Lab;
- Windows installer/portable build, Linux x64/ARM64 and macOS Intel/Apple Silicon packages.

### Beta limitations

- Windows binaries are not Authenticode-signed;
- macOS builds are ad-hoc signed but not notarized;
- native platform QA and the full PixInsight/WBPP compatibility matrix are ongoing;
- always review calibration assignments before processing valuable data.

## Italiano

Questa è la prima beta pubblica di AstroProject Forge: un workspace multipiattaforma che trasforma acquisizioni FITS/XISF multisessione in progetti PixInsight WBPP verificabili.

### Incluso

- import FITS/XISF generico e analisi header-first;
- ricostruzione della notte astronomica oltre la mezzanotte;
- gerarchia filtro → sessione di configurazione → notte;
- Flat Epoch automatiche e link manuali su file, notti e sessioni;
- matching motivato Flat/Dark/Bias con Master Library prioritarie;
- statistiche di acquisizione ed export CSV/JSON;
- Grouping Keywords WBPP adattive ed export progetto verificato;
- Quality Lab opzionale e Master Library Lab indipendente;
- installer/portable Windows, pacchetti Linux x64/ARM64 e macOS Intel/Apple Silicon.

### Limiti della beta

- i binari Windows non hanno ancora firma Authenticode;
- le build macOS sono firmate ad-hoc ma non notarizzate;
- QA nativa e matrice completa PixInsight/WBPP sono ancora in corso;
- controlla sempre le assegnazioni prima di elaborare dati importanti.
